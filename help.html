Getting started

  • Features

    1. View simple and easy recipes with TL;DRecipes. Log in to favorite and submit your favorite recipes.
    2. Click on the “;D” in the “TL;DRecipes” on the Home Page or Enter Konami code on the Home Page for yummy surprises.

  • How to Submit Recipes

    Click on the “Submit a Recipe” button at the top on the Navigation Bar

  • Dietary Restrictions

    1. V
      = Vegetarian
    2. Vg
      = Vegan
    3. Df
      = Dairy-Free
    4. K
      = Kosher
    5. Gl
      = Gluten-Free

Nav Bar

  1. Help Page
    Click on “Help” for a fun time
  2. Browse
    Click browse to search through the most popular recipes, recently submitted recipes or the recipes with the highest success rate
  3. Submit a Recipe
    Click submit a recipe to be taken to Submit a recipe page
  4. Search
    Toggle between recipe search and user search using the drop down box. Then, type in your search keyword and click “Search”
  5. Advanced Search
    Click on Advanced Search to be taken to the Advanced Search
  6. Create an Account/Sign In
    Click on create account or sign in to log into TL;DRecipes
  7. User Page
    Click on “User Page” to be taken to the User Page

Recipe Page

  1. Name of the Recipe
  2. Approximate time it takes to complete the recipe
  3. Percentage of people who have been successful making this recipe
  4. Dietary Restrictions of the recipe
  5. User who submitted the recipe
  6. Picture of completed Recipe
  7. List of Ingredients with quantity
    Click on to cycle through the alternative ingredients
  8. Instruction List
    You can also view a step by step instruction list, in case you’re on mobile. Click on the left or right arrow to navigate through the instructions
  9. Once you are done with the recipe, please tell us if you were successful with this particular recipe. It helps us filter out the not so successful ones and give you the recipes which are most important

Advanced Search Page

  1. User Search
    Search for a specific user by putting his/her username
  2. Recipe Search
    1. Search for a recipe with the exact phrase. No more, no less
    2. Search for a recipe whose name contains these words
  3. Ingredient Search
    1. Search for recipes containing these ingredients
    2. Search for recipes that do not contain these ingredients
  4. Prep Time
    Search for recipes with less than or equal to this completion time
  5. Dietary Restrictions
    Search for recipes with these dietary restrictions

User Page

  1. Name and picture of the user
  2. Additional information about user
    1. Number of submitted recipes
    2. Number of favorited recipes
    3. Date user joined TL;DRecipes
    4. Last date user logged in to TL;DRecipes
  3. A quick “About Me” of the user
  4. User’s submitted recipes
  5. User’s favorited recipes

Submit a Recipe Page(s)

  1. Step 1 - General Info

    1. Name of your recipe
    2. Upload recipe image
    3. Dropdown box of approximate total cook time (including prep time)
      1. Upload as many images as you want, but upload them one at a time
      2. You can also delete the pictures by clicking the x button below the picture
    4. Check box with the dietary restrictions. Select all that apply.

  2. Step 2 - Ingredient List

    1. Amount of ingredient necessary (ex ¼ cup, 1 head etc)
    2. Ingredient (ex Milk, Lettuce)
    3. Click “Add another ingredient” to add more ingredients

  3. Step 3 - Instructions List

    1. Type in the instructions. Make sure to keep it simple.
    2. Click “Add Step” to add a new step
    3. Click Finish to submit the recipe
